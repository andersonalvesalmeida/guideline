Botões principais

Botão class="" Descrição
btn Botão padrão para ações de baixa relevância. Esse estilo de botões poderá ser utilizado em outros locais sempre que a relevância for extremamente baixa, ou seja, não quiser chamar a atenção do usuário para a ação mas existe a necessidade da ação existir.
btn btn-primary Botões para ações primárias que devem ser feitas na interface.
btn btn-secondary Botão para ações secundárias

Botões auxiliares

Botão class="" Descrição
btn btn-warning O usuário deve ter atenção pois a ação pode influir outras áreas da aplicação.
btn btn-danger Indica que a ação que o usuário irá tomar pode impactar a utilização do sistema ou de algum componente específico.

Botões com ícone e nomenclatura

Botão Descrição
O botão de impressão será utilizado somente quando for especificado na aplicação. Sempre deve ser incluido no topo direito da aplicação
O botão de ajuda também será utilizado somente quando for especificado na aplicação, assim como o botão de impressão. Também deve ser incluido no topo direito e abrir uma modal com o texto relativo.
Botões de controle de controle de tamanho devem estar presentes em todas as páginas.
Botões que tem ícones, com exceção de "imprimir" e "ajuda", deverão possuir tooltip e não possuem nomenclatura.

Botões para ações

Como convenção, os botões só deve ser utilizado para ações enquanto hiperlinks devem ser utilizados para os objetos. Por exemplo, "Download" deve ser um botão, enquanto "atividade recente" deve ser um link.

Estilos de botão pode ser aplicado a qualquer coisa com a classe btn. Aplicada. No entanto, normalmente você vai querer aplicá-las as âncoras apenas e elementos do tipo button .

Compatibilidade

Como uma boa prática, tente sempre utilizar o tipo correto de botão para garantir uma renderização correta em diversos browsers. Se você tem um form , utilize um <input type="submit" class="btn"> como botão para submeter o formulário.

Desabilitados

Para botões desabilitados, utilize a classe .disabled para desabilitar botões em formado de âncora e o atributo disabled para elementos do tipo button .

Tipos de formulários

Nossa biblioteca possui 4 tipos diferentes de organização de formulários.

Nome Classe Descrição
Vertical (default) .form O rótulo do campo fica acima do controle, sempre alinhado à esquerda. É o formato padrão de formulários que vão dentro de janelas modais e espaços reduzidos.
Inline .form-inline Rótulo e controle ficam na mesma linha, alinhados à esquerda.
Busca .form-inline Formulário que já vem com um padrão estético de um campo de busca
Horizontal .form-horizontal labels com float right alinhados à direita, na mesma linha dos controles.

Exemplos de formulários

Formulário vertical

Utilizar em janelas modais

Para formulários inseridos em janelas modais, devemos utilizar a classe .form-vertical. Utilizando essa classe os campos do formulários ficam dispostos verticalmente. Veja um exemplo clicando no botão abaixo:

            <form class="form">
                <fieldset>
                    <div class="form-group">
                        <label for="field1_input">Nome</label>
                        <div class="controls">
                            <input type="text" class="form-control" id="field1_input" name="field1" placeholder="Informe seu nome"/>
                        </div>
                    </div>
                    <div class="form-group">
                        <label for="field1_input">E-mail</label>
                        <div class="controls">
                            <div class="input-prepend input-append">
                                <span class="add-on"><i class="icon-envelope"></i></span>
                                <input type="text" class="form-control" id="field1_input" name="field1" placeholder="Informe seu email"/>
                            </div>
                        </div>
                    </div>
                    <div class="form-group">
                        <label for="field1_input">Percentual <span class="add-on">%</span></label>
                        <div class="controls">
                            <div class="input-prepend input-append">
                                <input type="text" class="form-control" mask="999,99"  id="field1_input" name="field1" placeholder="Informe o percentual"/>
                            </div>
                        </div>
                    </div>
                </fieldset>
            </form>

Formulário horizontal

Utilizar em páginas

Para formulários utilizados diretamente na página, sem utilização de componentes como janelas modais, utilize a classe .form-horizontal. Utilizando essa classe os campos do formulário foram dispostos com o seu label posicionado do lado esquerdo, enquanto o campo fica posicionado ao lado direito. Veja um exemplo abaixo:

            <form class="form-horizontal">
              <fieldset>
                <div class="form-group">
                  <label for="field1_input" class="col-sm-2 control-label">Nome</label>
                  <div class="col-sm-10">
                    <input type="text" class="form-control" id="field1_input" name="field1" placeholder="Informe seu nome"/>
                  </div>
                </div>
                <div class="form-group">
                  <label for="field1_input" class="col-sm-2 control-label">E-mail</label>
                  <div class="col-sm-10">
                    <div class="input-prepend input-append">
                      <span class="add-on"><i class="icon-envelope"></i></span>
                      <input type="text" class="form-control" id="field1_input" name="field1" placeholder="Informe seu email"/>
                    </div>
                  </div>
                </div>
                <div class="form-group">
                  <label for="field1_input" class="col-sm-2 control-label">Percentual <span class="add-on">%</span></label>
                  <div class="col-sm-10">
                    <div class="input-prepend input-append">
                      <input type="text" class="form-control" mask="999,99"  id="field1_input" name="field1" placeholder="Informe o percentual"/>
                    </div>
                  </div>
                </div>
              </fieldset>
            </form>

Formulário inline

@

Utilizar em formulários de filtro

Esse tipo de formulário é inserido utilizando a classe form-inline. Utilizando essa classe os campos do formulário são dispostos horizontalmente. Utilize esse tipo de organização em formulários de filtros, conforme o exemplo abaixo:

          <form class="form-inline" role="form">
              <div class="form-group">
                <label class="sr-only" for="nome">Nome</label>
                <input type="text" class="form-control" id="nome" placeholder="Informe o nome">
              </div>
              <div class="form-group">
                <label class="sr-only" for="email">Email</label>
                <div class="input-group">
                  <div class="input-group-addon">@</div>
                  <input class="form-control" type="email" placeholder="Informe o email">
                </div>
              </div>
              <div class="form-group">
                <label class="sr-only" for="senha">Senha</label>
                <input type="password" class="form-control" id="senha" placeholder="Senha">
              </div>
              <div class="checkbox">
                <label>
                  <input type="checkbox"> Ativo
                </label>
              </div>
              <button type="submit" class="btn btn-default">Enviar</button>
            </form>

Formulário de busca

          <form class="form-inline">
                <div class="form-group">
                    <label class="sr-only" for="nome">Nome</label>
                    <input type="search" class="form-control" id="nome" placeholder="Informe o nome">
                </div>
                <button type="submit" class="btn btn-default">Buscar</button>
            </form>

Campos especiais

Os campos especiais possuem funcionalidades adicionais aos tipos básicos.

Nome Tipo Aparência Descrição
Date Docs type="text" datepicker mask="date" Ao focar o campo, exibe um calendário para a seleção de uma data para preenchimento do texto.
Máscara Docs type="text" mask="decimal"
R$
Garante que o campo será preenchido com uma formatação específica.
Chosen Docs select chosen Incrementa ao funcionamento padrão do select um campo de texto para facilitar a busca de uma opção específica.
Chosen Multiplo Docs select multiple="true" chosen Exibe listas de seleção multiplas em formato de chosen.

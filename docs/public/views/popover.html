Referências

  • http://getbootstrap.com/javascript/#popovers

Conceito

Utilizado para organizar um conjunto de interações ou informações que complementam um objetivo maior do usuário para com a interface.

Utilização

  • O elemento que irá conter a ação do popover deverá conter o atributo data-[atributo].
  • Os atributos como título, conteúdo, orientação e etc podem ser informados atravez de Javascript ou data-[atributo] no HTML

Veja a documentação completa em http://getbootstrap.com/javascript/#popovers.

Atributos via data-attributes

		<a class="btn" id="btn-popover" href="#" 
			data-toggle="popover"
			data-trigger="hover"
			data-title="Título"
			data-content="Clique aqui para acessar"
			data-placement="right">	Clique
		</a>

Atributos via Javascript

		$("#btn-popover").popover({ 
			placement: 'right',
			trigger: 'hover',
			content: 'Clique aqui para acessar',
			title:'Título'
		});

Atenção O popover será disparada automaticamente quando o elemento possuir o elemento data-toggle="popover"

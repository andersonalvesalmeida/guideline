Em um accordion, deve-se abrir cada ítem por vez. Caso seja necessário abrir mais opções, deve ser utilizado o datatables com agrupamento por colunas.

Accordion básico

Aplicação básica do componente sem utilizar nenhuma configuração customizada, utilizando apenas marcação HTML.

Este exemplo não utiliza uma chamada em javascript específica pois utiliza o processo de injeção via data-api do bootstrap.

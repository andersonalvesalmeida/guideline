Usando bootstrap-tab.js

Ativar abas via javascript (cada guia precisa ser ativado individualmente):

$('#myTab a').click(function (e) {
  e.preventDefault();
  $(this).tab('show');
})

Você pode ativar abas individuais de várias maneiras:

$('#myTab a[href="#profile"]').tab('show'); // Select tab by name
$('#myTab a:first').tab('show'); // Select first tab
$('#myTab a:last').tab('show'); // Select last tab
$('#myTab li:eq(2) a').tab('show'); // Select third tab (0-indexed)

Marcação

Você pode ativar uma aba de navegação ou pílula sem escrever qualquer javascript simplesmente especificandodata-toggle="tab" ou data-toggle="pill" ou um elemento. Adicionando o nav e nav-tabs classes de uma aba ul irá aplicar o estilo guia bootstrap.

<ul class="nav nav-tabs">
  <li><a href="#home" data-toggle="tab">Home</a></li>
  <li><a href="#profile" data-toggle="tab">Profile</a></li>
  <li><a href="#messages" data-toggle="tab">Messages</a></li>
  <li><a href="#settings" data-toggle="tab">Settings</a></li>
</ul>

Métodos

$().tab

Ativa um elemento aba e recipiente de conteúdo. A aba deve ter data-target ou um href visando um container node no DOM. 

<ul class="nav nav-tabs" id="myTab">
  <li class="active"><a href="#home">Home</a></li>
  <li><a href="#profile">Profile</a></li>
  <li><a href="#messages">Messages</a></li>
  <li><a href="#settings">Settings</a></li>
</ul>

<div class="tab-content">
  <div class="tab-pane active" id="home">...</div>
  <div class="tab-pane" id="profile">...</div>
  <div class="tab-pane" id="messages">...</div>
  <div class="tab-pane" id="settings">...</div>
</div>

<script>
  $(function () {
    $('#myTab a:last').tab('show');
  })
</script>

Eventos

Evento Descrição
Show Este evento é acionado na aba guia, mas antes da nova aba ser mostrada. Use event.target and event.relatedTarget para direcionar a aba ativa e ativa então a aba anterior (se disponível), respectivamente.
shown Este evento é acionado na aba principal e depois que uma aba tem sido demonstrada. Use event.target e event.relatedTarget para direcionar a aba ativa e então ativa a aba anterior (se disponível), respectivamente. 
$('a[data-toggle="tab"]').on('shown', function (e) {
  e.target // activated tab
  e.relatedTarget // previous tab
})

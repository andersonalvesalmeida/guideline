DataTables

O componente de datagrid deve ser utilizado quando a exibição de dados necessitar de filtragem, ordenação e paginação dos dados. A paginação só deve ser aplicada quando o número total de registros for acima de 50. Com uma simples chamada é possível um datagrid como o exibido ao lado.

Propriedades mais comuns

Por padrão, o datagrid será apresentado com paginação, quantidade de registro por páginas e filtro. Para manipular a visualizacao destes atributos podemos usar:

"iDeferLoading": 0Retorna 0 linhas na primeira busca.

"processing": true: Para processar dados do lado do servidor

"serverSide": true: Para processar dados do lado do servidor

"searching": false: Desabilita o campo de busca

"lengthMenu": [ 50, 100, 200, 500, 1000 ]: Seta as opções de paginação

"pageLength": 100: Seta a opção de paginação padrão

"paging": false: desabilita a paginação, usado em relatórios com poucas linhas

Utilização

                $(document).ready(function() {
                    //colocamos o datatable em uma variável para utilização em funções externas
                    table = $('.dt')
                    //Altera o loading para o padrão da guideline
                    .on( 'processing.dt', function ( e, settings, processing ) {
                    if(processing){
                        notificar('information', 'Processando...');
                    }else{
                        $.noty.closeAll();
                    }
                    return false;
                    })
                    //INicia o Datatable
                    .DataTable({
                    "iDeferLoading": 0,
                    "processing": true,
                    "serverSide": true,
                    "searching": false,
                    "lengthMenu": [ 50, 100, 200, 500, 1000 ],
                    "pageLength": 100,
                    "ajax": {
                      //Solicita os dados a um arquivo externo
                      "url": "data.cfm",
                      //Podemos passar por aqui valores de filtros externos ao datatables
                      "data": function ( d ) {
                        d.anoInicio = $("#anoInicio").val();
                        d.anoFim = $("#anoFim").val();
                        }
                    },
                    //Exemplo de como colocar link em um registro
                    "rowCallback": function( nRow, aData, iDisplayIndex ) {
                      $('td:eq(0)', nRow).html('<a href="pagina?id='+aData[1]+'">' +
                          aData[0] + '</a>');
                      return nRow;
                    },
                    //Adicionando classes`nas colunas
                    "columns": [
                            null,
                            { className: "text-right cpf-cnpj nowrap" },
                            { className: "text-right" },
                            { className: "text-right" },
                            { className: "text-right currency nowrap" },
                            { className: "text-right currency nowrap" },
                            { className: "text-right currency nowrap" }
                        ],
                        "drawCallback": function( settings ) {
                            //Chamando funções externas
                            formatData();
                        }
                    }); 
                });

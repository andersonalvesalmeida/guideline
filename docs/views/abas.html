O conceito de abas poderá ser usado para dinamizar e facilitar a navegação entre as informações do mesmo gênero. As abas auxiliam a trazer as informações para o usuário de forma separada, facilitando o entendimento da mesma. Dessa forma o usuário poderá também localizar rapidamente a área que deseja visualizar, sem que para isso seja preciso utilizar muito o scroll para navegar em uma página complexa.

O que está incluído

Você poderá altenar entre conteúdos através do uso de abas, sendo elas: superior (padrão), direita, base, e esquerda.

Mudar entre esses estilos é fácil, e requer pouca alteração de código.

Fade nas abas

Para utilizar fade in, adicione .fade para cada .tab-pane.

Requerimentos plugin jQuery

Exemplo de Abas

Para fazer as abas, crie um .tab-pane com um ID único para toda aba e envolvida com .tab-content.

Estou na Seção 1. Agora estou na Seção 2. E claro, por último estou na Seção 3.

Para a direita ou esquerda abas alinhados, envolver o .nav-tabs e o .tab-content em .tabbable.

Marcação Simples

Usando abas exige uma div, um conjunto de abas, e um conjunto de painéis de abas de conteúdo.

    <div class="tabbable"> <!-- Only required for left/right tabs -->
      <ul class="nav nav-tabs">
        <li class="active"><a href="#tab1" data-toggle="tab">Section 1</a></li>
        <li><a href="#tab2" data-toggle="tab">Section 2</a></li>
      </ul>
      <div class="tab-content">
        <div class="tab-pane active" id="tab1">
          <p>I'm in Section 1.</p>
        </div>
        <div class="tab-pane" id="tab2">
          <p>Howdy, I'm in Section 2.</p>
        </div>
      </div>
    </div>

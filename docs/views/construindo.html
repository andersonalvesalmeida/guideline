Pré-requisitos

  1. Git
  2. npm - É instalado jundo com o Node
  3. Node
  4. Bower
  5. Grunt

Siga o processo de instalacão utilizando os links acima na ordem em que aparecem na lista.

Repositório GitHub

Faça um clone ou fork do projeto git@github.com:andersonalvesalmeida/guideline.git

Criando um build

  • Abra uma instância do terminal/cmd e vá para a pasta onde você clonou o projeto git@github.com:andersonalvesalmeida/guideline.git.
  • Instale as dependências do node utilizando o comando sudo npm install.
  • Instale as dependências do bower utilizando o comando bower update.
  • Execute na raiz do projeto o comando grunt build se você estiver utilizando uma plataforma Unix ou grunt build.cmd se você estiver no Windows para iniciar o build.

Sucesso! Verifique o diretório /guideline na raiz do projeto. diretórios contendo os arquivos da guideline estarão lá.

A versão não minificada desses arquivos encontram-se também nesses diretórios. Eles são melhores para o desenvolvimento pois são apenas concatenados.

Atenção Certifique-se de sempre utilizar a versão minificada dos arquivos em produção. Esses arquivos já foram otimizados para obter a melhor performance de carregamento.

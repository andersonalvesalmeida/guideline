Tem como objetivo esclarecer a informação pertinente a uma interação ou conteudo.Se molda na maioria dos casos como legenda da interface.

Utilização

  • O elemento que irá conter o tooltip deverá conter o atributo data-toggle="tooltip" e um title="Texto aqui".
  • Os demais atributos como animação, orientação e etc, podem ser informados atravez de Javascript ou data-attributes no HTML
  • Por padrão, o tooltip possui orientação TOP, podendo ser alterada com o atributo data-placement="top|bottom|left|right"

Atenção O tooltip será disparada automaticamente quando o elemento possuir o elemento data-toggle="tooltip"

Exemplo

			<a id="tooltip" href="#" data-toggle="tooltip" data-placement="top" title="Eu sou um exemplo!">	
			  Demonstração
			</a>

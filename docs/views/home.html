Essa Guideline esta sendo desenvolvida para ajudar a padronizar o estilo visual de nossos sistemas e também prover componentes de facil utilização aos desenvolvedores.

Essa documentação visa explicar como utilizar cada um dos componentes. Se um componente desejado ainda não existir na guideline o ideal é que seja adicionado para só então ser utilizado.

É importante seguir as recomendações visuais indicadas pela guideline para manter o padrão.

Sugestões de melhorias serão sempre bem vindas

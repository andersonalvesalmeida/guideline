Datepicker básico

Aplicação básica do componente com configurações padrão, construindo com javascript.

		$('.datepicker').datepicker({
		    format: 'dd/mm/yyyy',
		    todayBtn: true,
		    language: 'pt-BR',
		    autoclose: true,
		    todayHighlight: true
		});

Essa inicialização já esta implementada na guideline. Para utilizar o datepicker básico, basta incluir o atributo datepicker no input.

<input type="text" name="data" class="form-control datepicker" placeholder="dd/mm/aaaa"/>

Todos os datepicker devem utilizar o formato DD/MM/YYYY para formatação das datas. Para garantir este formato, é necessário o uso de máscaras e validação.
